The bobogna.pl website uses cookie technology. We use cookies based on your consent, except where they are necessary for the proper provision of the electronic service.

The following third-party cookies are used on the website:

  • Google Analytics – generating statistics and analysis for the purpose of optimising the website
  • Google AdWords / AdSense – displaying personalised advertisements

Privacy policies of third-party providers: Facebook, Google, Pinterest, YouTube.

Using our website involves sending requests to the server. Each request is recorded in the server logs and includes, among other things: IP address, date and time, browser and operating system information.

Data stored in server logs is not associated with specific individuals using the website and is not used for the purpose of identifying users. It serves solely as auxiliary material for website administration.
